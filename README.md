## Installation

machine is available on [crates.io](https://crates.io/crates/machine) and can be included in your Cargo enabled project like this:

```toml
[dependencies]
machine = "~0.1.0"
```

Then include it in your code like this:

```rust
#[macro_use]
extern crate machine;
```
